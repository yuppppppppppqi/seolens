# SEO Lens ブログ コンテンツ戦略

## 戦略概要

SEO Lens のプロダクトに関連するキーワードで、**低KD（難易度50以下）かつ一定のボリューム**があるキーワードを中心に、ピラー＆クラスター構造でブログを構築する。

### ターゲット市場の優先順位

| 優先度 | 市場 | 理由 |
|--------|------|------|
| 1 | US (en) | 最大ボリューム、CPC高い、データ豊富 |
| 2 | JP (ja) | KD低め、SEO対策系のボリューム大きい |
| 3 | KR (ko) | 韓国SEO市場は競合少なく参入余地あり |
| 4 | TW (zh) | 中華圏への入口、SEO情報の需要高い |
| 5 | DE/FR/BR | 欧州・南米はデータN/Aが多く=競合少ない |

## 多言語展開戦略

### 各市場での記事ローカライズ優先順位

US市場で成果が出た記事をベースに、以下の順で多言語展開する：

```
US (en) → JP (ja) → KR (ko) → TW (zh) → DE (de) → FR (fr) → BR (pt) → ES (es)
```

### 各市場で狙うべきローカルキーワード

**JP（日本）- KD低め、高ボリューム多数:**
- seo対策とは (9,900 / KD:21) → 入門記事
- キーワードプランナー (22,200 / KD:16) → ツール解説
- キーワードマップ (1,300 / KD:0) → ハウツー記事
- seo ai (2,400 / KD:11) → トレンド記事

**KR（韓国）- 未開拓で大きなチャンス:**
- 키워드 분석 (キーワード分析) → ツール解説
- 백링크 만들기 (バックリンク構築) → ハウツー記事
- 검색엔진 최적화 (検索エンジン最適化) → 入門記事
- seo 최적화 (SEO最適化) → トレンド記事

**TW（台湾）- 中華圏への入口:**
- 關鍵字 分析 (キーワード分析) → ツール解説
- 反向連結 (バックリンク) → 入門記事
- SEO工具 (SEOツール) → 比較記事
- 搜尋引擎優化 (検索エンジン最適化) → ガイド

**DE（ドイツ）- 欧州最大市場:**
- seo optimierung → 最適化ガイド
- seo analyse kostenlos → 無料ツール紹介
- suchmaschinenoptimierung → 包括的ガイド
- backlinks aufbauen → リンク構築ガイド

### リンクルール

1. **クラスター → ピラー**: 全クラスター記事からピラーページへ必ずリンク
2. **ピラー → クラスター**: ピラーページから関連クラスター記事へリンク
3. **クラスター間**: 同じピラー内のクラスター記事同士でリンク
4. **ピラー間**: 関連するピラーページ同士でリンク（例: SEO Audit ↔ SEO Tools）
5. **CTA**: 全記事にSEO Lensツールへの導線を設置
